# RobustMSA
Kaiser’s single-variable measure of sampling adequacy (MSA) is a very useful index for debugging inappropriate items before a factor analysis (FA) solution is fitted to an item-pool dataset for item selection purposes.

# Authors
Urbano Lorenzo-Seva & Natasha Tonge

email: urbano.lorenzo@urv.cat

# Measure of sampling adequacy (MSA) in R
Code to related to the paper: 

Lorenzo-Seva, U., & Ferrando, P. J. (2021). MSA: The forgotten index for identifying inappropriate items before computing exploratory item factor analysis. Methodology, 17(4), 296-306.

## Requirements
R >= 4.2

Packages: None

## Run
source("RobustMSA_v2.r")
